Mouser.js

A small-ish* JS library for clickable tours.

(Ooh, looks shiny)

Download GitHub

Demos

Move from one place...

...to another.

move('#to-place') // By Selector
move({left: 50, top: 125}) // By Coordinates
Click me!

Highlight an element to click

pulsateUntilClicked()

Say something

With a title

annotate("Hello there")
annotate({title: "Fancy Annotation",
content: "This is a fancy annotation",
position: "top"})

Easy styling with CSS

#styled-mouser {
  background: #049CDB
}
#styled-mouser .mouser-pointer {
  background: url('pointer.png')
  background-size: contain
}

Runs simultaneously

mouser1.move('#simult-place', -100, 0)
mouser2.move('#simult-place', 100, 0)

Usage

  1. Install Mouser.js + dependencies

    <script src="jquery.js"></script>
    <script src="jquery.transit.js"></script>
    <script src="bootstrap.tooltip.js"></script>
    <script src="bootstrap.popover.js"></script>
    <script src="mouser.js"></script>

  2. Initialize a Mouser

    mouser = new Mouser();
    mousers = $('#mouser').mouser()

  3. Add a storyboard and you're done!

    mouser.move('h1').pulsateUntilClicked()
